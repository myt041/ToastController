# ToastController
Toast controller on debug and release mode

Step 1. Add the JitPack repository to your build file

      allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	  }
	  
Step 2. Add the dependency

 
        dependencies {
	        implementation 'com.github.myt041:ToastController:0.0.1'
	   }
